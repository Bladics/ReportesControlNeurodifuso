Problema 2

Existen otros tipos de funciones de membresía difusas. A las funciones membresía usuales (funciones del universo de discurso al intervalo ), se les conoce como funciones de membresía tipo-I o conjuntos difusos tipo-I. Cuando no se tiene precisión sobre el grado de membresía se puede pensar en que se conoce el valor inferior y superior de membresía posibles, en este caso se tendrán funciones de membresía evaluadas en un intervalo, como lo muestra la Figura 1, (Ross, 2017).
En esta figura, para un elemento particular el grado de membresía en un conjunto difuso A(), es expresado por el intervalo de membresía [] . Este tipo de funciones de membresía son nombradas en la literatura como conjuntos difusos tipo-II.
Así, todo lo que conoces hasta ahora sobre lógica difusa puede ser extendido utilizando conjuntos difusos tipo-II. El siguiente enlace: Type-2 Fuzzy Inference Systems es una breve descripción de como se realiza en MATLAB inferencia difusa utilizando funciones de membresía tipo-II.
En clase has revisado la implementación de un FIS tipo-I al siguiente ejemplo. Reproduce la implementación que se encuentra en la misma página de dicho FIS utilizando funciones de membresía tipo-II. Adjunta un enlace a tu página html donde se encuentre la implementación.
Creación de sistema FIS tipo I
Creando el sistema tipo 1
fis1=sugfis;
%agregando variables de entrada al sistema
fis1=addInput(fis1,[-1 1],'Name','E');
fis1=addInput(fis1,[-1,1],'Name','delE');
Agregando 3 funciones de membresía a cada entrada, funciones uniformemente distribuidas de tipo triangular. Los nombres son negative (n),zero(z) y positive(p)
fis1 = addMF(fis1,'E','trimf',[-2 -1 0],'Name','N');
fis1 = addMF(fis1,'E','trimf',[-1 0 1],'Name','Z');
fis1 = addMF(fis1,'E','trimf',[0 1 2],'Name','P');
fis1 = addMF(fis1,'delE','trimf',[-2 -1 0],'Name','N');
fis1 = addMF(fis1,'delE','trimf',[-1 0 1],'Name','Z');
fis1 = addMF(fis1,'delE','trimf',[0 1 2],'Name','P');
Graficando las funciones de membresía de las variables de entrada.
figure
subplot(1,2,1)
plotmf(fis1,'input',1)
title('Input 1')
subplot(1,2,2)
plotmf(fis1,'input',2)
title('Input 2')
Agregando la variable de salida a fis1.
fis1 = addOutput(fis1,[-1 1],'Name','U');
Agregando las funciones de membresía a la variable de salida. Las funciones son de tipo constante. Los nombres son negative big(NB), negative medium (NM), zero (z), positive medium (PM) y positive big(PB).
fis1 = addMF(fis1,'U','constant',-1,'Name','NB');
fis1 = addMF(fis1,'U','constant',-0.5,'Name','NM');
fis1 = addMF(fis1,'U','constant',0,'Name','Z');
fis1 = addMF(fis1,'U','constant',0.5,'Name','PM');
fis1 = addMF(fis1,'U','constant',1,'Name','PB');
Agregando reglas al FIS. Estas reglas crean un control proporcional de la superficie.
rules = [...
"E==N & delE==N => U=NB"; ...
"E==Z & delE==N => U=NM"; ...
"E==P & delE==N => U=Z"; ...
"E==N & delE==Z => U=NM"; ...
"E==Z & delE==Z => U=Z"; ...
"E==P & delE==Z => U=PM"; ...
"E==N & delE==P => U=Z"; ...
"E==Z & delE==P => U=PM"; ...
"E==P & delE==P => U=PB" ...
];
fis1 = addRule(fis1,rules);
Graficando la superficie de control.
figure
gensurf(fis1)
title('Control de superficie del FIS tipo-1')
Construcción del sistema FIS tipo-2.
Convertir el sistema FIS tipo-1 en un sistema FIS tipo-2.
fis2 = convertToType2(fis1);
Los sistemas de tipo-2 Sugeno usan funciones de membresía tipo-2 para las variables de entrada y tipo-1 para las variables de salida. Se deben definir las huellas de incertidumbre(FOU) de las funciones de membresía de entrada. Para lograr esto se debe establecer el factor de escalamiento para las funciones de membresía inferiores.
scale = [0.2 0.9 0.2;0.3 0.9 0.3];
for i = 1:length(fis2.Inputs)
for j = 1:length(fis2.Inputs(i).MembershipFunctions)
fis2.Inputs(i).MembershipFunctions(j).LowerLag = 0;
fis2.Inputs(i).MembershipFunctions(j).LowerScale = scale(i,j);
end
end
Graficar las funciones de membresía de entrada de tipo-2.
figure
subplot(1,2,1)
plotmf(fis2,'input',1)
title('Input 1')
subplot(1,2,2)
plotmf(fis2,'input',2)
title('Input 2')
La huella de incertidumbre (FOU) agrega incertidumbre al sistema y produce control no lineal sobre la superficie.
figure
gensurf(fis2)
title('Control surface of type-2 FIS')
